사용방법)


1. 로그인
  - 아이디는 실제 이메일 주소로 하셔야 하고, 비밀번호는 6글자 이상이어야 합니다.
  - 아이디, 비밀번호 등록 후 해당 메일 주소로 전송된 인증 메일로 인증하셔야 로그인할 수 있습니다.
2. 회원정보 입력
 - 주소 정보는 '복현동'으로 입력하시는걸 추천드립니다.
   (다른 주소 입력해도 괜찮지만, 샘플 데이터가 부족해서 제대로 기능 확인이 힘듭니다)

- 구현 기능(client ver)
1. 로그인/회원가입
2. 이미지 데이터(추천 스타일 등) 로딩(서버연동)
3. 남/여 구분하여 스타일 추천
4. 회원 주소에 해당하는 주변 미용실 목록 로딩
5. 헤어스타일 의뢰(주변 미용사들에게 의뢰서 전송), 입찰한 스타일리스트 입찰정보 로딩
6. 회원정보 수정

- 구현 기능(stylist ver)
1. 로그인/회원가입
2. 이미지 데이터(추천 스타일 등) 로딩(서버연동)
3. 내 미용실 정보 로딩(스타일리스트 ver)
4. 헤어스타일 입찰(주변 고객들의 의뢰 입찰)
5.  회원정보 수정


- 미구현 기능
1. 예약하기
2. 내 입찰 내역 리스트
3. 1:1채팅 기능
